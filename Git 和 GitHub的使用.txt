1、首先是先下载 Git 分布式版本控制系统

如果需要将自己的一个项目从本地 放到 GitHub远程仓库中保存起来，需要有以下的步骤
	a、git init   // 初始化一个Git仓库 .git的文件夹（隐藏的）
	b、git add .  // . 是当前目录 用于将当前目录下的所有文件和文件夹的更改添加到暂存区
		git add 文件名字 添加单独的文件
	c、git status // 在没有 b 指令之前所有需要添加的文件都是红色的字体
				  // 在执行 b 指令之后多有需要添加的文件都是绿色的字体
	d、git commit -m "这次更新的简短说明（自己输入）"

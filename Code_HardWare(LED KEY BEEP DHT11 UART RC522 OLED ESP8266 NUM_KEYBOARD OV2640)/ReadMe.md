## **项目名字：**

   基于物联网的智慧安防门禁系统设计与实现

*************************************************************************************************

## **项目简介**：

​      通过设置锁屏，输入初始化密码进入主界面，主界面会有相关功能模块的图标，点击相关图标，将会开启相应的能，通过屏幕边缘左滑右滑实现回退操作，返回上一步，能够在项目的主界面上实时显示当前的温湿度等环境因子。在每一个功能模块添加用户注册登录的界面，和服务器进行通信，服务器会进行账号密码匹配，同时还结合科大讯飞的离线命令语音进行客户端开发板和Ubuntu服务器进行通信。

*************************************************************************************************

## 项目内容功能：

​    1、画板功能模块  
​        功能简介：
​            实现不同画笔大小 颜色的切换 
​    2、相册展示模块
​        功能简介：
​            缩略图分页显示图片，点击小图，可以权展示该图，特效随机展示大图的展示过程
​    3、计算器模块
​        功能简介：
​            实现计算器的相关功能，能够检错
​    4、视频播放器模块
​        功能简介：
​            实现视频的 快进快退 进度条点击哪里播放哪里 视频的音量调节 暂停播放 倍速播放 上下滑动切换视频
​    5、环境监控模块
​        功能简介：
​            展示相关环境因子实时数据 温度 湿度 海拔 气压 烟雾浓度 等超过相关阈值会报警（蜂鸣器响LED灯亮）并且超过阈值的数据会标红
​    6、语音模块
​        功能简介：
​            通过事先规定的ID和命令在服务器端进行录制语音文件，在发送给服务器，服务器会根据客户端所发的
​            音文件进行识别，并且返回相应的ID，共服务器进行相关操作
